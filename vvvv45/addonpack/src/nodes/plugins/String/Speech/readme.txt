 their license

You need to install it in order to use this plugin.

http://www.microsoft.com/DOWNLOADS/details.aspx?familyid=5E86EC97-40A7-453F-B0EE-6583171B4530&displaylang=en#filelist

The File to download is: SpeechSDK51.exe

A copy of the Microsoft SAPI 5.1 EULA is included in this download.

To install, just copy these two files in the Plugins folder of vvvv:
-Interop.SpeechLib.dll
-Speech.dll


Note: To check if the speech engine is initialized properly, you can check the the node Status Pin 
(visible only in the Inspektor).
