Convex Hull 3d node for vvvv.

This node computes the 3d hull for a set of points, outputting vertices and indices.

The code is a port from java to c# from 
the excellent quickhull algorithm from John E. Lloyd (many thanks to authorise me to use his code).

http://www.cs.ubc.ca/~lloyd/index.html

To install, just copy the dll in the vvvv plugin folder
